# MojoDesktop


WebViewVer for [Mojo Frontend](https://github.com/SpikeHD/MojoFrontend).

------

## Feature

 + 快速打开 MojoFronted （按 T 开启，Esc 关闭）

## 使用方法
  1. 下载解压
  2. 启动原神 （必须要无边框方式启动，并且分辨率为 1920*1080） 
  3. 以管理员身份启动程序
  4. 输入 服务器ip 认证的key 进行连接
  5. Enjoy it！
